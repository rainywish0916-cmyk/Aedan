Title: Automated Python Portfolio Platform
Date: 2026-07-10
Category: Projects
Summary: A custom automated web portfolio powered by Python and complied directly through GitHub Actions workflows.

### Project Overview
I wanted a live portfolio that operates entirely on its own without requiring computer hosting. I configured a custom site framework using Python utilities.

### Key Features Implemented
* **Cloud Automation:** Integrated custom YAML build jobs to compile site elements seamlessly on code pushes.
* **Cottagecore Style:** Tied semantic layout rules directly to remote CSS styling mirrors for an organic reading layout.
* **Zero Maintenance:** The application relies entirely on serverless flat-file hosting architectures.
